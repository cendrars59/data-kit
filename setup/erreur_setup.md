# Erreurs fréquentes

Il arrive que des erreurs surviennent lors de l'installation. 

Nous listons dans ce document les erreurs les plus fréquentes. N'hésitez pas à contribuer ! 

## Mac OSX


Message d'erreur après le lancement de la commande : $ brew install python
Message d'erreur : python-2.7.10_2 already installed, it's just not linked

Resolution : Lancer la commande $ brew link --overwrite python
